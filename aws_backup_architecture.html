üöÄ Key Improvements Over Original Design

Event-Driven Architecture: Instead of polling, use S3 events to trigger SQS messages when files are uploaded.
Direct S3 Upload: Mobile app gets presigned URLs from Lambda to upload directly to S3, reducing API Gateway costs.
Queue-Based Processing: SQS ensures reliable message delivery to your Windows service.
Metadata Separation: Store file metadata in DynamoDB, actual files in S3.
Auto-Lifecycle: S3 lifecycle policies automatically delete files after X days.
Authentication: AWS Cognito for secure user authentication and authorization.

üí∞ Cost Optimization Strategies

S3 Intelligent Tiering: Automatically moves files to cheaper storage classes
Multipart Upload: For large videos, enables resumable uploads and better performance
Compression: Compress images before upload to reduce storage costs
Regional Selection: Choose AWS region closest to you for lower data transfer costs
